WSL 的基本命令

Windows Subsystem Linux

以下命令均在 PowerShell下运行。

  • 查看帮助
wsl --help
  • 列出可用的Linux发行版
wsl --list --online
  • 列出已安装的Linux发行版
wsl --list --verbose  
# 或 
wsl -l -v
  • 将目录更改为主页
wsl ~
# 也可
cd ~
  • 更新 WSL
wsl --update
  • 检查 WSL 状态
wsl --status
  • 关闭WSL， 默认情况下会占用很多内存，所以不用的情况下可以关闭掉

另外之前也提到过，如果要更改内存使用限制， 需要修改 .wslconfig 文件，在用户目录下。

wsl --shutdown

当然，如果需要打开直接在 PowerShell下输入 wsl 即可开启。
